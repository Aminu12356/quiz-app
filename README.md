# Quiz App 🧠

An interactive quiz application built with HTML, CSS, and JavaScript.

## Features
- Multiple choice questions
- countdown timer
- Score tracking and results display
- Progress bar
- Responsive design

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## How to Use
1. Click "Start Quiz"
2. Select your answer for each question
3. View your final score at the end
